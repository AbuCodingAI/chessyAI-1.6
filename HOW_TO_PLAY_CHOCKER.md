## üîß Troubleshooting

### "Python not found"
**Solution:** Install Python from https://python.org/

### "Module 'chess' not found"
**Solution:** 
```bash
pip install chess python-chess
```

### "Stockfish not found"
**Solution:** 
1. Download from https://stockfishchess.org/
2. Place in `stockfish/` folder
3. Or update path in `chocker.py` line 28

### "Permission denied"
**Solution:** Run as administrator or check file permissions
