## Troubleshooting

### "Cannot find batch files"
**You don't need batch files!** Just:
1. Open http://localhost:3000
2. Select AI from dropdown
3. Play!

### "Server not running"
Run this command:
```bash
node server.js
```
Then open: http://localhost:3000

### "Want to start fresh"
Double-click: `START_CHESSY_1.4.bat` in the root folder

## Tips for Playing Against Chessy 1.4

### It's Strong!
- Chessy 1.4 is GM level (~2700 ELO)
- It will punish mistakes
- It sees 10+ moves ahead in tactical positions

### How to Challenge It
1. **Play solid openings** - Don't make early mistakes
2. **Avoid tactics** - It will find all combinations
3. **Play positionally** - Long-term plans work better
4. **Be patient** - It takes 10-30 seconds per move

### If It's Too Hard
Try these AIs instead:
- **Master (2000)** - Strong but beatable
- **IM (2500)** - Tough challenge
- **Chessy 1.3 (2500)** - Slightly weaker than 1.4

### If It's Too Easy
You're probably a GM! Try:
- **Super GM (2700)** - Maximum Stockfish
- **Random Guy** - Shows ELO 1, plays 3400 (troll AI)
