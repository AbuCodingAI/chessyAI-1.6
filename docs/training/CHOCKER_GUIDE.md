## The Rules

### 0. ULTIMATE DISRESPECT OPENING (NEW!)
**As White:**
1. f3 (Fool's Mate setup - ULTIMATE DISRESPECT)
2. If Black plays d5: g4 (Grob's Attack)
   Else: Kf2 (Bongcloud variation)

**As Black:**
1. If White plays d4: f6 (Barnes Defense)
   If White plays e4: e5 (normal response)
2. If played e5: Kf7 (Bongcloud)
   If played f6: g5 (NO Kf7)

### 1. EN PASSANT IS MANDATORY (HOLY HELL!)
If en passant is available, Chocker MUST play it. No exceptions. It's the law.

### 2. STALEMATE IS PRIORITY
If Chocker can force a stalemate from a winning position, it will. Maximum troll achieved.

### 3. THROW WHEN WINNING
When up +3 or more, Chocker plays the ABSOLUTE WORST move possible. Not "bad but safe" - the WORST.

### 4. GM MODE WHEN LOSING
When losing or close, Chocker switches to 2800 ELO Stockfish depth 20. No mercy.

### 5. UNDERPROMOTION ONLY (with exception)
- **ONLY promote to queen if it causes stalemate** (ultimate disrespect!)
- If rook causes stalemate: Underpromote instead (bishop/knight)
- +5 or more: Bishop promotion (ultimate disrespect)
- +3 to +5: Knight promotion (still disrespectful)
- Never queen (unless stalemate)

### 6. RAGE MODE (THE SECRET WEAPON)
**If opponent doesn't take en passant when available:**
- All rules suspended (except en passant is still forced)
- Chocker goes FULL 3000 ELO Stockfish depth 25
- No more throws, no more stalemates
